Build Instructions

1.Go to build folder
2.Enter command :  make all
3.To clean Object files of previous run, command : make clean
4.After executing step 2 run ./main to run the executable

Note
1.All header files are stored in the include folder
2.All Implementations are in the src folder
3.main.cpp is at root folder
4.books folder has all books and it is here index.txt will be made and stored
